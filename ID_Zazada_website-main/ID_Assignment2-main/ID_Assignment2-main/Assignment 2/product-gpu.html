GTX 1660 Ti

The Nvidia GeForce GTX 1660 Ti is a powerful, mid-tier, Turing-based, desktop GPU. The GTX 1660 Ti relies on the TU116 chip with 1536 shader units, 48 ROPs, 96 TMUs and a 192-bit-wide memory bus (clocked at 6 GHz, effectively 12 GHz, bandwidth: up to 288 GB/s).
